Project: E-Commerce Backend API

Description:
A Node.js and Express backend for an e-commerce platform, using MongoDB with Mongoose. The API supports categories, products, shopping cart management, and order checkout with server-side price calculation.

Included functionality:
- Category CRUD operations
- Product CRUD operations with filtering, sorting, field selection, and pagination
- Cart endpoints for adding, updating, removing, and clearing items
- Order creation from cart, order retrieval, and status updates
- Centralized error handling and request sanitization
- Seed script to populate sample data

Tech stack:
- Node.js
- Express
- MongoDB / Mongoose
- dotenv

Run instructions:
1. Clone the repository
2. Install dependencies: npm install
3. Create .env from .env.example and set MONGO_URI
4. Optional: seed the database with npm run seed
5. Start the server with npm start
